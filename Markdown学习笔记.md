# Markdown学习笔记

## 1.标题写法

前置加“ # ”，一级标题加一个，二级标题加两个，依此类推...（注意不要忽略掉“ # ”后面的空格）

## 2.字体

```
*斜体文本*
**粗体文本**
***粗斜体文本***
```

## 3.分割线

连续的三个以上的“ * ”或“ - ”

## 4.下划线

```
<u>带下划线文本</u>
```

## 5.脚注

```
[^要注明的文本]
```

后注与前注有所不同

## 6.代码框

- 连续三个“ ~ ”

- 对于单个代码框可以“ ~ ”..." ~ "链接

## 7.区块

​	“ >  "+文字，多层嵌套的话就用多个” > "

## 8.链接

这是一个链接< 网址 >

这是一个链接[链接名]（网址）
